# vue-multiplePage

> vue多页面配置方案（研究时为了重构公司官网）

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8180
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
